# 🚀 Chrome Extensions Collection

This repository contains a collection of **Chrome Extensions developed for learning, experimentation, and productivity**.

Each project in this repository demonstrates different features of the **Chrome Extensions API (Manifest V3)** and focuses on building practical browser tools with modern UI and functionality.

The goal of this repository is to explore and build **useful browser extensions using JavaScript and the Chrome Extensions platform**.

---

## 📦 What You Will Find Here

This repository includes various Chrome extension projects such as:

* 🚫 **YouTube Ad Blocker Extension**
  Automatically skips ads and removes promoted content on YouTube.

* 🚀 **Developer New Tab Extension (Apna Dev Tab)**
  A custom new-tab dashboard for developers with tools, todos, focus tracking, and theme switching.

* ⚙ **Utility Chrome Extensions**
  Small tools built to explore Chrome APIs such as storage, tabs, scripting, and UI customization.

More extensions may be added in the future as part of ongoing development and learning.

## 🚀 How to Run an Extension (Developer Mode)

1. Open **Google Chrome**
2. Navigate to:

```id="extrepo2"
chrome://extensions
```

3. Enable **Developer Mode**
4. Click **Load Unpacked**
5. Select the extension folder you want to run

Example:

```id="extrepo3"
youtube-adblocker-pro
```

The extension will now appear in your Chrome toolbar.
