# 🚀 Apna Dev Tab

A modern **Developer Productivity Chrome Extension** that replaces the default browser new tab with a customizable developer dashboard.

Apna Dev Tab helps developers stay focused, manage tasks, and quickly access frequently used developer tools.

---

## ✨ Features

### ⏰ Smart Clock & Date

* Displays **live time and current date**
* Automatically updates every second

### 👋 Personalized Greeting

* Shows greeting based on time (Morning / Afternoon / Evening)
* Users can save their **name locally**

### 🎯 Daily Focus

* Set your **main goal for the day**
* Helps developers stay productive

### 📝 Todo List

* Add todos by pressing **Enter**
* View and manage your daily tasks
* Toggle visibility of todo section

### 🚀 Developer Tools Launcher

* Save frequently used developer tools like:

  * GitHub
  * Stack Overflow
  * ChatGPT
  * VS Code Web
  * CodePen
  * LeetCode
* Add custom tool URLs

### 🎨 Theme Switch

* Toggle between **Light Mode / Dark Mode**

### 🖼 Custom Background

* Upload background image
* Set background via **image URL**

### 👁 Section Visibility Control

Users can toggle sections like:

* Developer Tools
* Todo List

## 🚀 Installation (Developer Mode)

1. Open **Google Chrome**
2. Go to:

```id="devtab2"
chrome://extensions
```

3. Enable **Developer Mode**
4. Click **Load Unpacked**
5. Select the project folder

```id="devtab3"
apna-dev-tab
```

Now open a **New Tab** to see the custom dashboard.

---

## 🧠 How It Works

The extension uses **localStorage** to save:

* User name
* Todos
* Developer tools list
* Daily focus
* Background image
* Theme preference

This ensures data persists across browser sessions.
